Imagens Flexíveis

Usaremos as Tags PICTURE e SOURCE para adaptar o tamanho das imagens
em cada tamanho de tela

Usamos a Tag IMG dentro da tag PICTURE

A Tag SOURCE tem 3 parâmetros:

  • type: Indica o media type
  • srcset: Configura o nome da imagem que será carregada quando atingir o tamanho indicado
  • media: indica o tamanho máximo a ser considerado para carregar a imagem indicada no atributo srcset

            
                Existe uma ordem entre os SOURCE: 

                Os item mais a cima sejam os menores tamanhos para max-width
                e que os seguntes sejam maiores, de forma crescente. O último item dentro
                de PICTURE deve ser a imagem padrão.

Áudio

Usamo a Tag AUDIO para para tocar qualquer áudio no site
Temos os seguintes parâmetros para a AUDIO:
  1. Preload: indica se o áudio será pre-carregado ou não. Tem três atributos:
    • metadata - vai carregar apenas informaçções sobre os arquivos
    • none - não vai acrregar nada até que dê play ou um script faça rodar
    • auto(padrão) - vai carregar o arquivo de audio inteiro, mesmo que o usuario nunca dê play
  2. Controls - apresenta player na tela. Caso não colocado, o usuario não consegue interagir
  3. Autopay - vai iniciar a reprodução do audio assim que a pagina foir carregada
  4. Loop - vai fazer com que o audio seja repetido eternamente assim que terminar a sua reprodução

Video

Usamos a Tag VIDEO para incrementar vídeos no nosso site
Principais parâmetros:
  • width - indica a largura do vídeo que vai aparecer na tela
  • poster - configura uma imagem que vai aparecer como uma capa, enquanto não da play
  • controls - vai configurara se os controles do vídeo vão aparecer
  • autoplay - diz para o navegador se vai tocar automaticamente assim que a pagina carregar
