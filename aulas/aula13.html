Aplicando Cores ao Nosso Site

Podemos usar os padrões:

  • RGB( )
  • RGBA( )
  • HSL( )
  • HSLA( )
  • HEX( )

Como representar um Cor

As telas são compostas por 3 cores primárias:
Vermelho, verde e azul
Cada cor primária pode ter um valor entre 0 e 255, totalizando 256 possibilidades.

Usando gradientes em CSS

Para fazer um degradé usamos o parâmetro:
BACKGROUND-IMAGE: LINEAR-GRADIENT(90deg, cor 1, cor 2);
Podemos usar graditente radial e com mais de 2 cores!
