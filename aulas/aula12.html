Trabalhando com Estilos - Introdução às CSS

CSS inline style

Técnica mais básica de aplicar estilos, usamos dentro de parâmetros de HTML5.
Usamos o parâmetro STYLE para aplicar um estilo.

Esse é um teste usando CSS Inline Style

Esse título também contém estilo inline

Esse título não contém nenhum estilo;
Usar inline funciona apenas para a Tag onde foi adicionado!

CSS internal style

Adiciona estilos de forma mais dinâmica e prática, podemos adicionar uma Tag STYLE dentro do HEAD do nosso documento local.

Esse título h5 foi alterado pelo inernal style!
Vericar no HEAD caso tenha dúsivas!

As configurações pontuais (inline style) vão prevalecer sobre as config. gerais (internal style).

CSS external style

Usar estilos externos mantém o arquivo mais organizado e reaproveitamos os estilos em outras paginas de nosso site de forma mais eficientes.
Usamos a Tag LINK especialmente configurada para trabalhar com arquivos externos de estilo. Essa Tag deve ser colocada dentro da área HEAD do HTML.

Dentro da área do HEAD usamos link:css para linkar ao arquivo externo da CSS.
Ao criar o link:css, em href podemos criar o arquivo com a ajuda do VS Code usando Ctrl+clique.
@charset "UTF-8"; - Regra charset caso haja problemas com acentuação.
/* */- comentário em CSS

Recomendações:

  • CSS Externo - use sempre que puder
  • CSS Interno - use para pequenas configurações
  • CSS inline - procure evitar
                É possível misturar as 3 técnicas, criando um CSS externo para as configurações globais,
                CSS interno para configurações locais de um documento e CSS inline para pequenas configurações pontuais.
