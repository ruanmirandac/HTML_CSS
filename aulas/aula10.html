Links de Direcionamento Externo

Dentro da Tag A podemos usar o parâmetro TARGET
Valores para target:

  • _blank - abre link em nova janela em branco
  • _self - abre link na janela atual (padrão)
  • _top - desfaz todos os frames e abre o link no navegador completo
  • _parent - siilar ao uso do _top em uma referência a janela mãe
  • nome-do-frame - caso esteja usando frames

Destino do Link

Usamos o Parâmetro REL para indicar a natureza do destino do link
Valores para rel:

  • next - indica que o link é para a proxima parte do doc
  • prev - indica que o link é para a parte anterior do doc
  • author - indica que é um link para o site do autor do artigo
  • external - indica que é um link externo ao seu site
  • nofollow - indica que é um link para um site não endossado, como link pago
